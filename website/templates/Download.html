System Requirements

Because of limitations built into the ocaml binaries (specifically ocamldebug), Oasis IDE requires a unix-based operating system like Mac OS X or Linux. Additionally, it requires java, ocaml, ocamlc and ocamldebug.

Download Oasis IDE

The most recent version of Oasis IDE is 1.0beta. If you're a Brown student, we recommend using the installation on the department machines because it will be updated regularly with every bugfix we push to the repository. See @brown for how to run Oasis at Brown University.

Executable .jar file

You may download Oasis IDE as a compiled .jar file. Download the current version: OasisIDE-1.0beta.jar. To run the jar and start the application run the following:

$ java -jar OasisIDE-1.0beta.jar

Compile from source

The source code for Oasis IDE is publicly available on github.com. You can clone or download the source code from github and compile the source yourself. An ant build file is included in the repository.

$ git clone git://github.com/jbowens/Oasis-IDE.git
$ cd Oasis-IDE
$ ant compile
